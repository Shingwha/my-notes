# 高效学习方法总结

## 🎯 核心原则

### 1. 主动学习
- 不要被动接收信息
- 提出问题并寻找答案
- 用自己的话复述所学内容

### 2. 间隔重复
- 制定复习计划
- 利用艾宾浩斯遗忘曲线
- 定期回顾旧知识

### 3. 实践应用
- 学以致用
- 项目驱动学习
- 教是最好的学

## 📚 具体方法

### 费曼技巧
1. 选择一个概念
2. 向小孩解释它
3. 发现理解盲区
4. 简化并重新组织

### 番茄工作法
- 25分钟专注学习
- 5分钟休息
- 4个番茄后长休息

### 思维导图
- 中心主题
- 分支扩展
- 颜色标记

## 📈 学习路径规划

```
基础概念 → 实践项目 → 深入研究 → 分享输出
```
